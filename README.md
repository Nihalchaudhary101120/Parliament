# 🏛️ Parliament Battle

🚀 **Live Demo:** https://parliamentbattle.aalsicoders.in/

---

## 🎮 About the Game

**Parliament Battle** is a strategic multiplayer board game where players compete using tactics, cards, and dice mechanics to dominate the parliament.

This project blends **game logic, real-time interaction, and modern web development** to create an engaging experience.

---

## ⚔️ Game Features

* 🎲 Dice-based movement system
* 👥 Multiplayer support (up to 6 players)
* 🧠 Strategy-driven gameplay
* 🃏 Special cards:

  * 🔬 **Scientist Card** → Increases weapon damage (+3%)
  * 🛠️ **Engineer Card** → Boosts Parliament HP
  * 🕵️ **Agent Card** → Reduces enemy damage by 50% (next turn)
* 🏠 Home & battle tile mechanics
* 💥 Dynamic attack system

---

---

## 🧠 How the Game Works

1. Players join the game room
2. Game starts when all players are ready
3. Players roll dice to move
4. Landing on tiles triggers:

   * Attack ⚔️
   * Card effects 🃏
   * Strategic decisions 🧠
5. Use cards smartly to survive and dominate
6. Last surviving / strongest player wins 🏆
